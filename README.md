# ChickenRun
2020-2 컴퓨터그래픽스 프로젝트 : Unity3D 게임 제작

<h3>프로젝트 개요</h3>
- 장애물을 피해서 목표 지점까지 도달하는 3D 러닝 게임<br>
- 닭🐓이 모든 병아리🐥를 모아서 목표 지점에 도착하는 것이 목표<br>
